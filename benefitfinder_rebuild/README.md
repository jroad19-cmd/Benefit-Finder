# Benefit Finder Verified PA

This is a clean Next.js starter app focused on hard eligibility gates.

## Key fixes
- Results recompute immediately from profile input changes.
- PA-only programs are excluded when state is not PA.
- Housing and income hard gates exclude ineligible results.
- Multiple Myeloma matches blood-cancer/general-cancer resources, not lung-cancer-only resources.
- Includes PA Homestead/Farmstead Exclusion (Act 1 of 2006) as a separate program.

## Deploy
Upload the contents of this folder to your GitHub repo root, then deploy with Vercel.
